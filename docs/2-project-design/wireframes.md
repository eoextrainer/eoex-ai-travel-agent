# Project Design: Wireframes

## User Journey
- Home: Form to input origin, destination, date, budget, preferences.
- Results: Flight offers, accommodations, transport, food, shopping, places to visit.
- Booking: Confirmation step (placeholder for future booking integration).
- Admin Dashboard: Table with filters for user, destination, budget.

## Screens
- Search Form: Simple grid with inputs and submit.
- Results List: Cards for offers with key info.
- Admin Dashboard: Data table with filter inputs and pagination.
